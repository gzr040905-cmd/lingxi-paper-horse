# 临溪纸马铺

原创民俗悬疑网站小游戏。玩法是阅读页面、寻找可疑中文关键词、在搜索框中检索并解锁隐藏档案。

## 打开方式

直接打开 `index.html` 即可游玩。

如果浏览器限制本地功能，可以在本目录启动一个本地静态服务后访问：

```powershell
python -m http.server 8766
```

然后打开：

```text
http://127.0.0.1:8766/
```

## 给别人玩

看 [DEPLOY.md](DEPLOY.md)。

- 最稳妥：发布到 Netlify、GitHub Pages、Vercel 或 Cloudflare Pages，生成公网链接。
- 临时试玩：同一 Wi-Fi 下用局域网 IP 访问。
- 离线分享：把整个 `lingxi-folk-mystery` 文件夹压缩发给别人。

## 基本规则

- 每次只输入一个连续中文关键词。
- 不支持空格、英文、数字或标点。
- 初始推荐关键词：`沈婉`、`河门`、`纸马`。
- 页面编号从 `01/24` 到 `24/24`，编号越大越接近真相。
- 已发现页面会保存在浏览器本地记录里，可以用页面上的“清空”按钮重置。
