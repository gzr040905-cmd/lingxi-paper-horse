# 临溪纸马铺

原创民俗悬疑网页小游戏。玩家以沈知遥的身份进入“临溪纸马铺”留下的旧档案网站，通过单个中文关键词检索，逐步查明三年前临溪河祭背后的身份替换案，并判断沈婉是否仍然活着。

## 打开方式

直接打开 [index.html](C:/Users/86133/Documents/New%20project/lingxi-folk-mystery/index.html) 就能玩。

如果要更稳定地测试离线缓存、手机访问或安装体验，建议在目录内启动一个本地静态服务，再访问：

`http://127.0.0.1:8766/`

## 玩法规则

- 每次只输入一个连续的中文关键词。
- 搜索命中后，只会显示一个档案入口；真正打开后才算阅读进度。
- 提前猜中后期词也不能跳关，必须先满足前置证据。
- 连续搜索失败两次会给方向提示，四次会给更窄的范围提示。
- 进度保存在浏览器本地，可用页面右侧“清空”按钮重置。

## 目录说明

- `index.html`：页面结构与移动端/桌面端入口
- `style.css`：三栏调查台、移动底栏、证据视觉样式
- `story-data.js`：24 份档案数据、人物与地点资料
- `visuals.js`：证据图片映射
- `assets/evidence/`：证据视觉素材
- `service-worker.js`：离线缓存
- `docs/story-bible.md`：剧情圣经
- `docs/clue-map.md`：24 页因果链与关键词顺序

## 公开发布

当前结构适合直接发到 GitHub Pages 或 Netlify。因为是纯静态站点，不需要后端，不会把你的本机账号信息带到网站里。
